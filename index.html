Redirect-Expander

Expands an path into a url and redirects to said url.

Path: "http://this.host/foo.example.com:8080/bar?one=2&two=%(RANDOM)%" redirects to "http://foo.example.com:8080/bar?one=2&two=7" (or whichever random number is given)

NOW
Takes no argument and inserts current time in milliseconds
ONEOF
Takes an argument list, separated by ';' and inserts one of these
eg. %(ONEOF:horse;cow;dog;cat)% will expand to a 4 legged mammal
Remember %3B expands into a ';'
RANDOM
Takes a comma separated list of 0, 1 or 2 arguments:
0
Inserts a number between 0 and 9 (both included)
1
Inserts a number between 0 and the argument (both included) (can be negative)
2
Inserts a number between the two arguments (both included) (can be negative)
